Support Program for Statistical Analysis of Count of Motifs in Temporal Graphs
Randomization of the time value of a temporal graph, with subsequent re-evaluation of the count of motifs.

IL PROGRAMMA RICHIEDE L'UTILIZZO DI UN SISTEMA OPERATIVO LINUX

1) INSERIRE IL PROPRIO PATH NELLA VARIABILE project_path
PRESENTE IN MAIN.C: IL PATH DEVE ESSERE UGUALE A QUELLO CHE
SI USEREBBE PER ACCEDERE A QUESTA CARTELLA DA TERMINALE. IL
PATH DEVE ESSERE PRECEDUTO DA cd.

2) PER ESEGUIRE IL PROGRAMMA DIGITARE "MAKE RUN" DA TERMINALE
DALLA CARTELLA FILE_MANIPULATION_NEW
(DOPO AVER COMPILATO IL MAKEFILE SEMPRE NELLA STESSA CARTELLA)

3) E' NECESSARIO COMPILARE ANCHE TUTTI I MAKEFILE PRESENTI
NELLA CARTELLA MOTIFS_SIGNIFICANCE. MAGGIORI INFORMAZIONI
NEI READ_ME PRESENTI SEMPRE NELLA STESSA CARTELLA.
IN PARTICOLARE SI CONSIDERANO I MAKEFILE PRESENTI IN
code_paper/examples/exact

4) SI CONSIGLIA LA MODIFICA DEL NUMERO DI PERMUTAZIONI SE SI
VUOLE RIDURRE IL TEMPO DI ESECUZIONE, SOPRATTUTTO PER DATASET
MOLTO ESTESI (OSSIA TUTTI ECCETTO CollegeMsg.txt)

ULTERIORI INFORMAZIONI:

  • Nella cartella "motifs_significance/datasets" sono disponibili
    tutti i dataset che possono essere usati per l'esecuzione
    del programma.
  • Nella cartella "Result_Every_Motif" sono contenuti i risultati
    del programma per ogni tipo di motif.
  • Nella cartella "motifs_significance/code_paper/examples/motifs"
    sono presenti tutti i file rappresentanti i motif.
  • All'interno di "file_manipulation_new/src e file_manipulation/mnp_lib"
    sono presenti file header e c, che appartengono al programma di
    randomizzazione SFMT (già sviluppato). Il resto dei file sono
    stati creati da me.
  • La documentazione doxygen è presente in "file_manipulation/doc /html/index.html"
